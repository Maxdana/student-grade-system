# Student Grade System

Практична робота №5 з курсу **Основи програмування мовою JavaScript**.

Це проста програма-журнал, в якій можна зберігати студентів, предмети та їх оцінки.  
Програма дозволяє додавати студентів, виставляти оцінки та переглядати середній бал.

---

# Що вміє програма

- додавати студентів (ім'я та email)
- додавати предмети
- виставляти оцінки студентам
- показувати журнал оцінок
- рахувати середній бал студента
- рахувати середній бал по предмету
- показувати статистику по групі

Журнал зберігається у localStorage, тому після перезавантаження сторінки дані не зникають.

---

# Структура даних

## Об'єкт Student

Кожен студент зберігається як об'єкт.

Властивості:

- id
- name
- email
- grades

Приклад структури:

grades = {
  Математика: [10, 9, 8],
  Історія: [7, 9]
}

---

# Методи студента

У класі Student є такі методи:

- addGrade(subject, score) — додати оцінку
- getGrade(subject) — отримати оцінки по предмету
- getAverageGrade() — порахувати середній бал

Також використовується getter для GPA.

---

# Управління студентами

Для роботи з усіма студентами використовується клас GradeManager.

Основні методи:

- addStudent(student)
- removeStudent(id)
- updateStudent(id, data)
- getStudent(id)
- addStudentGrade(studentId, subject, score)

---

# Статистика

Програма також може показувати:

- середній бал групи
- середній бал по предмету
- топ студентів
- студентів з низькими оцінками

---

У програмі використано:

- вкладені об'єкти (grades)
- математичні обчислення середніх значень
- сортування студентів за балами
- збір статистики по всіх студентах

---

# Тести

Для перевірки роботи методів створено прості unit tests.

Вони знаходяться у файлах:

tests.html  
tests.js

Щоб запустити тести, потрібно відкрити tests.html і натиснути кнопку запуску.

---

# Як запустити програму

1. Відкрити файл index.html у браузері
2. Додати студентів та предмети
3. Додавати оцінки
4. Переглядати журнал та статистику
5. програма зберігає дані, а також може очистити журнал
